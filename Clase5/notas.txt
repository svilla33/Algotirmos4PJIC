explicacion del algoritmo:

1. tomamos la expresion infija y la recorremos de izquierda a derecha
2. a cada numero o operador ( + - * / ( ) )  de la infija lo llamamos token
3. los + - tienen precedencia 1 y los * / tiene precedencia 2, es como una caracteristia de "peso"
4. si el token es un numero pasa a la salida
5. si el token es un operador y la pila esta vacia metalo a la pila
6. si el token es un operador y la pila no esta vacia
